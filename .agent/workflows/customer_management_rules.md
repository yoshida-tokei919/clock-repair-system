# 顧客登録・管理ルール

## 1. プレフィックス(管理番号接頭辞)の義務化
- **全ての顧客にプレフィックスを必須**とする。
- **業者(B2B)**: 登録時に作業者が「半角英数字(3文字以内)」で指定する（例: JK, YSD）。
- **個人(B2C)**: システムが自動的に **`C`** を割り当てる。
- 修理番号はこのプレフィックスを用いて `[Prefix]-[連番]` (例: JK-001, C-001) として発行される。

## 2. 入力制限とバリデーション
- **使用可能文字**: 漢字、ひらがな、カタカナ、半角英数字、長音(ー)、一部の記号（括弧など）のみ。
- **特殊記号の禁止**: 表記ゆれを防ぐため、特殊な記号や全角英数字の使用を制限する。
- **プレフィックス**: **半角英数字のみ**。

## 3. 重複登録の防止
- 以下のいずれかが既存顧客と一致する場合、保存前に警告ダイアログを表示し、作業者に確認を促す。
  1. **屋号・社名(またはお名前)** が完全一致。
  2. **プレフィックス** が完全一致。
  3. その両方が一致。
- 警告が出た場合、既存顧客への紐付け（修正）か、重複を承知での新規登録かを選択する。

## 4. 業者(B2B)登録の優先事項
- 業者の場合は「屋号・会社名」を必須入力とし、最優先の管理項目とする。
- 「担当者名」は任意項目とする。
