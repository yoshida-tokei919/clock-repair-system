# Task 037: B2B共有ページ CSV/メモ出力機能

## 目的

B2B共有ページで、取引先が案件情報をCSVやメモ用テキストとして外部ツールへ持ち出せるようにする。

用途:

- Excel
- Googleスプレッドシート
- 自社DB
- FileMaker
- Notion
- kintone
- Appleメモ
- Google Keep
- LINEメモ
- メール

## 背景

現在、B2B共有ページは複数案件を1URLで確認できるようになっている。

実装済み:

- EstimateDocument単位の共有URL
- 複数Repair表示
- アコーディオンUI
- コメント
- 店舗返信
- 承認
- 差戻し
- statusLog
- B2B/B2C分岐
- エンドユーザー説明文コピー
- 貴社専用メモ
- スマホ可読性改善

次に、B2B先が案件情報を自社管理へ持ち出しやすくする。

## 基本方針

- DB保存しない
- 当店には送信しない
- 一括出力設定はページ上部
- 各案件カード内には出力設定UIを置かない
- 出力項目を選択可能にする
- CSVコピー
- メモ用テキストコピー
- Web Share APIでメモアプリ共有
- Web Share API非対応環境ではクリップボードコピー
- ご案内金額は空欄でも出力できる
- 共有ページの既存機能を壊さない

## 禁止ワード

UIや出力文言に以下を使わない。

- 掛け率
- 掛率
- マークアップ率
- 利益
- 利益額
- 利益率
- 粗利
- マージン
- 差額

使用してよい表現:

- 当店見積額
- ご案内金額
- +¥36,000
- ×1.50 相当

## 対象候補ファイル

- `src/app/customer/repairs/[token]/page.tsx`
- `src/app/customer/repairs/[token]/CustomerRepairActions.tsx`
- `src/app/customer/repairs/[token]/CustomerRepairAccordion.tsx`

必要なら小さな新規componentを追加してよい。

例:

- `src/app/customer/repairs/[token]/CustomerExportTools.tsx`

ただし不要な分割は禁止。

## UI仕様

ページ上部に折りたたみ式のカードを置く。

見出し:

```text
貴社用データ出力設定
```

説明文:

```text
出力項目を選択して、CSVやメモ用テキストとして利用できます。
この内容は当店には送信されません。
```

チェック項目候補:

- 案件番号
- 顧客名
- ブランド
- モデル
- Ref
- 当店見積額
- ご案内金額
- ステータス
- 連絡先
- 対応メモ

ボタン:

- CSVをコピー
- メモ用テキストをコピー
- メモアプリへ共有

## CSV出力イメージ

```csv
案件番号,顧客名,ブランド,モデル,Ref,当店見積額,ご案内金額,ステータス,連絡先,対応メモ
Y-003,佐藤恒一,ROLEX,Datejust,16233,72000,,未確認,,
```

ご案内金額は空欄でよい。

## メモ用テキスト出力イメージ

```text
Y-003 / 佐藤恒一 様
ROLEX Datejust Ref.16233

【当店見積額】
¥72,000

【ご案内金額】
¥

【ステータス】
未確認

【連絡先】
TEL:

【対応メモ】
```

## Web Share API方針

スマホでは `navigator.share()` を使う。

注意:

- 保存先アプリは指定できない
- title は渡せるが、実際の保存名は受け側アプリ依存
- 非対応環境ではコピーにfallback

- 電話履歴
- 電話番号DB保存
